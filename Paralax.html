Aktuelle Uhrzeit

Anzeige der aktuellen Uhrzeit.

<h7 id="Uhrzeit"> </h7>

// Überschrift h7 mit ID Uhrzeit

h7{
background-color: #302F2F;
font-family: ‚Gotham, "Helvetica Neue", Helvetica, Arial, "sans-serif";
color: #fff;
font-size: 50Px;
padding: 10Px;
}

// Überschrift 7
// Hintergrundfarbe

function time() {
var d = new Date();
document.getElementById (Uhrzeit") .innerHTML = d.toLocaleTimeString();
}
setInterval(time, 1000);

// Funktion time
// Variable d gibt die aktuelle Zeit an
// Ausgabe der ID Uhrzeit mit der Angabe im Zeitcode

// Angabe im Intervall von 1000 Millisekunden also einer Sekunde
